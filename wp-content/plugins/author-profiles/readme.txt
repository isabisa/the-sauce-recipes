== Description ==

The plugin as name suggest display list of authors in your website using widget area section.  
This plugin is very simple for any user to show the author in the sidebar. Simply, just drag the author widget into the sidebar and 
the Wordpress Author Profile Avatars List plugin show the author in sidebar. 

The Plugin used the tabular format system in which you can add any number of columns depending upon the sidebar width.You can exclude the some authors which you do no want to display in the sidebar widgets just by typing the author name separated by comma. eg:sam,sabir in the column.

For further information or if you have any questions/installation please do not hesitate to contact us on our website: http://themefantasy.com/plugin/wordpress-author-plugin-widget/
